# Project Management MVP - Kanban Studio

A Project Management App featuring a Kanban board with drag-and-drop card management and an AI chat sidebar to create, edit, and move cards.

## Features

- User sign-in
- Kanban board with renameable columns and drag-and-drop cards
- AI chat sidebar to manage board content (create, edit, move cards)
- Dark/light mode toggle

## Tech Stack

- Frontend: Next.js 16, Tailwind CSS v4, @dnd-kit
- Backend: Python FastAPI, SQLite
- AI: OpenRouter
